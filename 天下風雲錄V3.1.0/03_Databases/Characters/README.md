# 03_Databases/Characters

本目錄保存《天下風雲錄》所有正式人物資料。

原 `03_OfficialCharacterFiles/` 已依 Appendix E Repository Guidelines 遷移至本目錄。

---

## 職責

- 保存主角、重要 NPC、歷史人物、江湖人物與勢力人物的正式資料。
- 記錄角色身份、背景、能力、關係、狀態、伏筆與登場紀錄。
- 作為小說、遊戲與互動敘事的人物依據。
- 對應 Story Bible、Official Databases、Runtime 與 Gameplay。
- 保存 Character Database 規格與索引。

---

## 規則

1. 人物檔不得與 Story Bible 衝突。
2. 人物資料應可追溯、可更新、可交叉引用。
3. 角色狀態改變後，應同步更新人物檔與 Runtime。
4. 重要角色應建立獨立 `.md` 檔案。
5. 遊戲中的即時狀態應放入 `02_Runtime/`，正式人物設定仍以本目錄為準。
6. 歷史人物不得違反重大歷史框架。
7. 本目錄不得建立第二套人物 Canon。
