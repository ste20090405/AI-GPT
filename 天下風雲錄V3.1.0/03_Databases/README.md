# 03_Databases

本目錄保存《天下風雲錄》Official Databases。

---

## 職責

Official Databases 用於實作 Story Bible 所定義之正式資料。

包含：

- Characters
- World
- Maps
- Timeline
- MartialArts
- Court
- Glossary

---

## 原則

1. 一種資料，一個資料庫。
2. Story Bible 為最高依據。
3. Database 不得反向修改 Story Bible。
4. Database 與 Runtime 保持同步。
5. 所有資料應可追溯、可驗證。

---

## Canon Level

Official Databases / System Canon
