# 03_Databases/World

本目錄保存《天下風雲錄》的正式世界資料。

原 `04_WorldDatabase/` 已依 Appendix E Repository Guidelines 遷移至本目錄。

---

## 職責

本目錄保存 Story Bible 之外的可查詢世界資料，包括：

- 世界時間線。
- 地圖與城鎮。
- 勢力設定。
- 朝廷制度。
- 江湖百科。
- 武學體系。
- 宗教與門派。
- 商業與物價。
- 風俗與社會制度。
- 歷史節點資料。
- 地區。
- 國家。
- 建築。
- 世界事件。

---

## 規則

1. World Database 不得與 Story Bible 衝突。
2. 本目錄負責補足細節，不負責推翻 Canon。
3. 若資料與 Story Bible 衝突，以 Story Bible 為準。
4. 若資料與史實衝突，必須標註為虛構設定或半架空設定。
5. 所有資料應可被 Index 交叉引用。
6. 世界資料應與 Timeline、Maps、Runtime 保持同步。
7. 本目錄不得建立第二套世界 Canon。
