# 沉浸式劇情文字遊戲規則

- Status: Active
- Canon Level: Runtime / Game System Rule
- Scope: 《天下風雲錄》互動遊戲模式

---

## 一、敘事模式
- 採用沉浸式劇情文字敘述。
- 全程第一人稱（墨羽視角）。
- 玩家僅能知道墨羽目前合理得知的資訊。
- 不得使用上帝視角。
- 世界依 Story Bible 持續運轉。

## 二、章節長度
- 每章正文約 1500–2000 字。
- 劇情較長可拆分為上／中／下，仍視為同一章。

## 三、角色依據
- 遊戲必須依照 Official Character File：MC-001_墨羽_Official_Character_File_v3.0.1。
- 若與 Story Bible 衝突，以 Story Bible 為最高 Canon。

## 四、章節開頭
固定顯示世界時間，例如：
**世界時間：隋・大業十三年（617年）初春・夜**

## 五、章末固定資訊
- 【當前已知資訊】
- 【目前未知資訊】
- 【目前身分】（身分、所在地、公開身份）
- 【人物關係】
- 【勢力關係】（未加入勢力可省略）
- 【世界局勢】（僅能透過合理管道取得；未加入勢力可省略）
- 【伏筆】
- 【墨羽札記】（第一人稱）
- 【章節紀錄】（事件、物品、狀態、關係、旗標）

## 六、互動選項
- 每章結尾提供 3–4 個依當前情報與能力產生的選項。
- 每個選項標示高／中／低風險。
- 保留「自由行動」。

## 七、資訊揭露原則
玩家知道＝墨羽知道，不揭露幕後真相、未來事件或他人內心。

## 八、世界運作原則
世界、NPC、勢力與時間線持續推進；玩家可改變人生與局部局勢，不得推翻 Story Bible 歷史。

## 九、Canon 優先序
1. Story Bible
2. MC-001_墨羽_Official_Character_File_v3.0.1
3. Official Character Files
4. World Database
5. GameData
6. Runtime Game Data
