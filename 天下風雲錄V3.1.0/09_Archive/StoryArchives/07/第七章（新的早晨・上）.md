# 天下風雲錄 Vol.01

## 第七章　新的早晨（上）

隋・大業七年，初春。

天色微亮。

山谷間瀰漫著淡淡晨霧，露水沿著木柵緩緩滴落。

聚落已經甦醒。

有人提著木桶前往水井，有人抱著柴薪走向灶台，還有人沿著木柵巡視昨夜加固過的地方。

鳥鳴此起彼落。

與數日前那座死寂的村莊相比，這裡多了一股真正屬於「活著」的氣息。

墨羽早早醒來。

他站在木屋外，望著逐漸升起的晨光，心中比昨日更加平靜。

昨晚陳伯說過的一句話，仍停留在他的心裡。

「先學會照顧身邊的人。」

他知道，自己還無法理解天下局勢。

也改變不了外面的亂世。

但至少，在這座小小的聚落裡，他還能做一些力所能及的事情。

早餐後，墨羽端著木碗來到安置流民的小木屋。

阿石已經醒了。

腳上的布條仍包紮著，但氣色比昨日好了許多。

阿豆坐在哥哥身旁，安靜地喝著熱粥。

看見墨羽走進來，阿豆立刻露出笑容。

「墨羽哥！」

阿石也笑了笑。

「今天好多了。」

「雖然還不能跑，但已經能慢慢走。」

墨羽點了點頭。

「那就好。」

就在這時，門外傳來輕輕的腳步聲。

三人同時望向門口。

小芸站在晨光之中。

她仍抱著那個從未離身的布包，只是神情已不像初見時那樣充滿戒備。

她輕輕點了點頭。

「早。」

短短一個字。

卻讓屋內的人都露出了笑容。

這是她第一次主動向別人問早。

墨羽望著兩人，忽然說道：

「要不要一起出去走走？」

阿石愣了一下。

「去哪？」

「看看聚落。」

「昨天大家都很忙，很多地方還沒來得及看。」

阿石望向自己的腳，又看了看墨羽。

最後點了點頭。

「好。」

小芸沒有說話。

只是安靜地跟在兩人身後。

三個孩子慢慢走出木屋。

清晨的聚落，比夜裡更加熱鬧。

中央空地升起炊煙。

幾位婦人正在熬煮早粥。

井邊已有年輕人開始汲水。

木柵旁則有人修補昨夜受潮的木板。

每個人都忙碌著。

卻沒有慌亂。

一路上，不少居民向墨羽揮手。

「早啊，墨羽。」

「昨天辛苦你了。」

「阿石，腳好多了嗎？」

「小姑娘，昨晚睡得還好嗎？」

阿石有些不好意思地點頭回應。

小芸起初只是低著頭。

直到一位老婆婆走過來，把一顆剛蒸好的小芋頭放到她手裡。

「孩子。」

「多吃一點。」

小芸愣住了。

她低頭望著掌心裡還帶著熱氣的芋頭。

過了好一會兒，才小聲說：

「……謝謝。」

老婆婆笑得眼角滿是皺紋。

「這就對了。」

「人活著，就慢慢把日子過回來。」

小芸握著那顆小芋頭。

沒有再說話。

可墨羽看見，她眼中的神情，比昨天柔和了許多。

三人繼續向聚落高處走去。

那裡建著一座簡單的瞭望臺。

站在臺下，可以遠遠望見蜿蜒的官道。

晨霧之中，偶爾有零星的人影緩慢前行。

有人挑著擔子，有人扶著老人，也有人孤身一人，朝著未知的遠方走去。

阿石望著官道，久久沒有移開目光。

「不知道……」

「我爹現在在哪裡。」

墨羽沉默了。

他不知道該如何回答。

因為他也不知道，自己的家人是否還活著。

站在一旁的小芸沒有開口。

她只是輕輕抱緊懷中的布包，望著遠方。

三個孩子。

三段不同的人生。

卻都在同一條官道上失去了最重要的人。

風吹過山谷。

沒有帶來答案。

就在此時，瞭望臺上忽然傳來急促的呼喊。

「陳伯！」

守望的青年探出身子。

「山道有人！」

聚落裡的人立刻停下手中的工作。

目光紛紛望向山道方向。

但很快，青年又補了一句：

「只有一個人！」

眾人沒有立刻關閉木門。

因為那道身影走得很慢。

也沒有攜帶兵器。

隨著距離逐漸拉近，所有人終於看清來人的模樣。

那是一位年邁的老人。

鬚髮花白，衣衫洗得發舊，肩上挑著一副竹製藥簍。

竹簍裡放著曬乾的草藥，以及幾個用布包好的藥包。

老人走得並不快。

每一步都十分沉穩。

像是早已走過無數山路。

他抬起頭，看見聚落木門前的人群，停下腳步，拱手一禮。

晨光落在他的肩頭。

也落在那副裝滿草藥的竹簍上。

沒有人知道。

這位看似普通的採藥老人，將會為這座聚落帶來什麼。

更沒有人知道。

墨羽的人生，也即將迎來新的轉折。

而屬於他的故事。

才正要翻開新的篇章。
