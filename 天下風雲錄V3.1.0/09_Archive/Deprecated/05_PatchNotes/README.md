# 05_PatchNotes

本目錄保存《天下風雲錄》Official Patch Notes 與 Release Notes。

---

## 職責

- 保存正式版本更新說明。
- 保存官方修正與補充紀錄。
- 對應 Version History 與 Git Commit。
- 說明哪些更新已整合回 Story Bible 或 Official Databases。

---

## 原則

1. Patch Notes 只記錄更新，不取代 Story Bible。
2. Patch Notes 不保存正式設定正文。
3. 舊版 Patch Notes 保留作為歷史紀錄。
4. Patch Notes 應與 Versioning 保持一致。

---

Version Record
