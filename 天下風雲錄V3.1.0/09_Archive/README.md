# 09_Archive

Archive 用於保存《天下風雲錄》舊版、廢棄、備查或歷史資料。

---

## 職責

- 保存舊版設定。
- 保存已被新 Canon 取代的資料。
- 保存歷史備份與參考資料。
- 保存不再直接使用但仍需追溯的文件。

---

## 規則

1. Archive 不作為最新 Canon。
2. Archive 內容不得覆蓋 Story Bible。
3. 若引用 Archive，必須標明其為舊版或備查資料。
4. 正式資料應保留於對應正式目錄，不放入 Archive。
5. Git History 已能追蹤的細微變更，不必重複封存。
