# Save_001 / CharacterMemory

## 定位

本檔保存墨羽已認識、聽聞或接觸過的人物紀錄。

本檔不取代 Official Character File，也不保存墨羽尚未知曉的人物秘密。

---

```text
劉山（劉叔）：聚落護衛，性格沉穩，曾協助墨羽完成汲水。墨羽感到他可靠。

小芸：官道事件中的倖存女孩，父親已安葬。最初沉默戒備，後來向墨羽說出姓名。仍抱著布包，內容未知。

阿石：流民少年，照顧弟弟阿豆。腳傷已包紮，可慢行。與墨羽建立初步友誼。

阿豆：阿石年幼弟弟，抵達聚落時疲憊昏睡，已安頓。

白髮老婦人：聚落居民，曾提供熱粥與粗餅，肯定墨羽的付出。
```
