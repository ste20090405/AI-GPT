# CONTRIBUTING

## Workflow
Proposal → Story Bible → Runtime → Databases → Gameplay → Patch Notes → Archive

## Branch Naming
- feature/<name>
- fix/<name>
- docs/<name>
- refactor/<name>

## Commit Convention
Follow Appendix H Git Commit Convention.

## Pull Requests
- Keep Story Bible as the highest canon.
- Ensure cross references are updated.
- Update Patch Notes for official releases.

## AI Collaboration
AI-generated content must remain consistent with the Story Bible and Official Databases.
