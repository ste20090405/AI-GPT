# 02_OfficialPatchNotes

Official Patch Notes 用於整理《天下風雲錄》所有正式版本更新與補充說明。

---

## 職責

- 保存正式版本更新說明。
- 保存官方修正、補充、規則調整與資料庫更新記錄。
- 對應 Git Commit / GitHub History，方便追溯變更原因。
- 說明每次版本更新後，哪些資料已整合回 Story Bible 或對應資料庫。

---

## 規則

1. Patch Notes 只記錄更新，不取代 Story Bible。
2. 若 Patch Notes 與 Story Bible 衝突，以 Story Bible 為準。
3. 正式更新後，相關內容應同步整合回 Story Bible 或對應資料庫。
4. Patch Notes 不保存人物、勢力、武學等正式正文，只保存版本變更說明。
5. 舊版 Patch Notes 保留作為歷史紀錄，不覆蓋。

---

## 建議命名

```text
PatchNotes_v3.0.0.md
PatchNotes_v3.0.1.md
PatchNotes_v3.1.0.md
```
