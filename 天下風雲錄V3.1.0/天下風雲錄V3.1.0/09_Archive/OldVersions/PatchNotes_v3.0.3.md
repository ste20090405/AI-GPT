# 第一章 Story Bible 更新

## 【新增】作品核心

本作品核心正式修訂為：

> 英雄，因天下而誕生。

而非：

> 英雄改變天下。

---

## 【新增】最高精神

正式列為最高 Canon：

> 生而微末者，當真無聲嗎？

最終答案：

> 蒼生無言，俠為其聲。

注意：

不得提前完整揭示。

---

## 【修訂】第一世

第一世正式名稱：

> **亂世浮萍**

墨羽因戰亂失去生計。

不得不加入瓦崗軍。

加入原因：

> 求生。

不是：

- 忠義
- 革命
- 理想

---

## 【新增】李世民線

接受延攬原因：

長時間觀察：

- 治軍
- 治民
- 百姓
- 戰後善後

最後判斷：

此道路，

更能改善百姓生活。

# 第六章 Gameplay Rules 更新（v3.0.3）

本章由 `07_Staging/Gameplay/Updates/Gameplay_Rules_Update_v3.0.3.md` 正式整合而來，原 Staging 標記不再適用。

### Gameplay Core

- 新增第一人稱沉浸式敘事規範。
- 新增「玩家知道＝墨羽知道」資訊同步原則。
- 新增世界時間固定顯示格式。
- 新增 Gameplay Canon 優先序。

### Gameplay UI

- 新增每次遊戲回覆固定結構。
- 新增章末固定資訊排序。
- 新增 Runtime GameData 同步規則。

### Dynamic Information

- 新增【當前已知資訊】動態管理。
- 新增【目前未知資訊】動態管理。
- 新增【人物關係】動態管理。
- 新增【勢力關係】條件式顯示。
- 新增【世界局勢】條件式顯示。

### 墨羽札記

- 新增墨羽札記完整保存規則。
- 新增歷史札記自動調閱機制。
- 新增管理者模式完整調閱能力。

### Gameplay Modes

- 新增 GitHub 管理模式。
- 新增遊戲模式切換規則。
- 新增 Runtime 無縫接續規則。

---

## 【修正（Fixed）】

### 敘事規則

- 修正全程採墨羽第一人稱視角。
- 修正禁止上帝視角敘事。
- 修正不得透露墨羽未知資訊、幕後真相、他人內心、未來事件或隱藏情報。
- 修正所有劇情資訊必須符合 Story Bible、人物資料、世界資料與 GameData。

### 互動系統

- 修正每回合固定提供四個互動選項。
- 修正第四項固定為「自由行動」。
- 修正每個選項皆須標示風險與影響。
- 修正不得產生墨羽目前不可能想到或無法執行的選項。

### Runtime

- 修正章末資訊更新方式。
- 修正人物關係同步規則。
- 修正世界局勢更新條件。
- 修正勢力資訊顯示條件。

---

## 【調整（Changed）】

### Gameplay Flow

- 每章建議長度調整為 1500–2000 字。
- 「上／中／下」改為平台回覆拆分，不再屬正式章節制度。
- 每次拆分仍須維持完整章末資訊與互動選項。

### Dynamic Information

- 已知資訊與未知資訊改為動態維護。
- 人物關係改為依劇情重要性顯示。
- 已完成、失效、過期或不再重要的資訊不再長期累積。

### 墨羽札記

- 不再限定夜晚才能新增札記。
- 改為墨羽具備休息、停留、書寫或整理思緒等合理條件時即可建立。
- 歷史札記調閱改依事件關聯，而非最近兩天。

### Runtime Synchronization

Gameplay Runtime 改為即時同步以下區塊：

- 【當前已知資訊】
- 【目前未知資訊】
- 【人物關係】
- 【勢力關係】
- 【世界局勢】
- 【伏筆】
