# 07_Documents/Templates

本目錄保存正式 Markdown 文件模板。

---

## 職責

- Character Template。
- Faction Template。
- Location Template。
- Martial Art Template。
- Event Template。
- Patch Notes Template。
- Runtime Template。

---

## 原則

1. Template 只提供格式，不保存正式設定。
2. Template 不得與 Story Bible 或 Appendix F 衝突。
3. Template 修改後應檢查既有正式文件是否需要同步調整。
