# Repository Purpose

本 Repository 為《天下風雲錄》唯一正式 Canon Repository。

主要負責保存與維護：

- Story Bible
- Runtime
- Official Databases
- Gameplay
- Official Patch Notes
- Project Rules
- Documentation
- Templates
- Development Tools
- Archives

Repository 不負責創造 Canon。

所有 Canon 皆源自 Story Bible。
