# 提案一：隱藏互動選項（⑤）

## 說明

1. 遊戲預設互動選項：
   - ①～③：一般劇情選項
   - ④：自由行動
   - ⑤（隱藏）：特殊觸發選項

2. ⑤（隱藏）平時不顯示，僅於符合指定條件時出現。

3. 觸發條件包含但不限於：
   - 墨羽身分
   - 公開身分
   - 年齡
   - 能力
   - 聲望
   - 人物關係
   - 勢力關係
   - 劇情旗標
   - 世界事件
   - 真實歷史事件
   - 歷史戰役
   - 地點
   - 時間線
   - Story Bible 或官方資料庫定義之條件

4. 可觸發：
   - 特殊劇情
   - 隱藏事件
   - 歷史事件參與
   - 戰役參與
   - 特殊人物相遇
   - 特殊勢力互動
   - 特殊任務

5. 必須遵守：
   - Navigation.md
   - Story Bible
   - Official Character Files
   - World Database
   - Official Patch Notes
   - 不得違反官方 Canon。

6. 所有選項格式固定：
   - 每個選項皆必須包含「風險」。
   - 每個選項皆必須包含「備註」。
   - 「風險」與「備註」為固定欄位，不得因一般選項或隱藏選項而改變，也不得省略。
