# 天下風雲錄 Gameplay 提案彙整

> 狀態：Staging / Proposal Summary
> 用途：彙整、分類、評估與追蹤 `天下風雲錄 提案資料.md` 中的 Gameplay 提案。
> Canon Status：Non-Canon / Staging Only

---

## 定位

本文件是「提案彙整與評估表」。

來源檔案：

```text
07_Staging/Gameplay/天下風雲錄 提案資料.md
```

本文件不保存所有原始提案細節，而是將提案整理為可測試、可評估、可轉入下一版 Official Patch Notes 的條目。

---

## 彙整規則

```text
1. 每筆彙整項目必須對應至少一個 Proposal ID。
2. 不得改變使用者原始提案意圖。
3. 可合併相近提案，但必須保留來源 Proposal ID。
4. 彙整後需標示分類、狀態、測試結果與版本建議。
5. Accepted 只代表建議採用，不代表正式 Canon。
6. 只有寫入 Official Patch Notes 後，才可成為正式規則。
```

---

## 彙整分類

```text
Interaction     互動選項與選擇設計
UI              顯示格式與提示方式
Roleplay        角色扮演提示與人格／執念
Rules           遊戲規則或系統流程
Narrative Flow  劇情推進節奏
Workflow        Staging / Patch Notes / 自動化流程
Other           其他
```

# 彙整紀錄

## SUMMARY-20260705-001：隱藏互動選項（⑤）系統

Related Proposal IDs:
- GP-20260705-001

Category: Interaction / UI / Rules
Status: Testing
Canon Status: Non-Canon / Staging Only
Target Version: v3.1.1 / Pending

### Summary

新增一個隱藏互動選項（⑤）。平時不顯示，僅當角色狀態、劇情旗標、人物關係、地點、時間線、世界事件或官方資料定義的條件成立時出現。

### Design Intent

讓遊戲能在不破壞一般選項平衡的前提下，呈現特殊劇情、隱藏事件、歷史事件參與、特殊人物相遇或特殊任務。

### Testing Notes

待後續實際遊戲場景測試：
- 是否能正確隱藏與顯示。
- 是否不提前劇透。
- 是否仍遵守 Canon。
- 是否不削弱 ④ 自由行動。

### Risks

可能造成玩家覺得有「最佳選項」或被系統誘導；需要明確保持 ⑤ 為特殊觸發，而非優勢選項。

### Recommendation

Testing。

### Patch Notes Candidate Text

```text
新增隱藏互動選項（⑤）：僅於符合特定條件時顯示，可觸發特殊劇情、隱藏事件、歷史事件參與或特殊人物互動。⑤ 不取代 ①～④，且不得違反官方 Canon。
```

## SUMMARY-20260705-002：互動選項人格／執念提示

Related Proposal IDs:
- GP-20260705-002

Category: Interaction / Roleplay / UI
Status: Testing
Canon Status: Non-Canon / Staging Only
Target Version: v3.1.1 / Pending

### Summary

劇情互動選項 ①～③ 產生時，比對 MC-001 墨羽 Official Character File。若選項符合墨羽真實人格或執念，於「備註」中加入提示，例如「符合善良」、「符合責任感」、「符合初心」。

### Design Intent

幫助玩家理解選項與墨羽角色性格之間的關係，同時保留玩家自由選擇。

### Testing Notes

待後續實際遊戲場景測試：
- 提示是否自然。
- 是否造成過度誘導。
- 是否破壞公平性。
- 是否與「風險」欄位並存良好。

### Risks

若提示過強，可能讓玩家誤以為該選項是正解。必須明確規定提示只作角色扮演參考，不代表最佳選項。

### Recommendation

Testing。

### Patch Notes Candidate Text

```text
互動選項 ①～③ 可於「備註」中顯示與墨羽人格或執念相符的角色扮演提示。此提示僅供理解角色傾向，不代表最佳選項，也不得影響玩家自由選擇。
```

## SUMMARY-20260705-003：Gameplay Proposal Capture Workflow

Related Proposal IDs:
- GP-20260705-003

Category: Workflow
Status: Recommended
Canon Status: Non-Canon / Staging Only
Target Version: v3.1.1 / Pending

### Summary

將 Gameplay 提案拆分為兩個檔案管理：

```text
07_Staging/Gameplay/天下風雲錄 提案資料.md
07_Staging/Gameplay/天下風雲錄 提案彙整.md
```

前者保存遊戲中提出的原始提案，後者負責分類、彙整、測試狀態與版本候選文字。

### Design Intent

避免原始提案、自動化規則、測試結果與版本更新候選文字混雜在同一文件中，提高長期維護性。

### Testing Notes

已初步建立檔案分工，後續每次提案時測試是否能正確寫入資料檔並同步更新彙整檔。

### Risks

若未同步更新兩份檔案，可能造成提案資料與彙整狀態不一致。

### Recommendation

Recommended。

### Patch Notes Candidate Text

```text
新增 Staging Gameplay 提案管理流程：原始提案寫入 `天下風雲錄 提案資料.md`，彙整與版本候選內容寫入 `天下風雲錄 提案彙整.md`。
```
