> 狀態：Draft
> 版本：v3.1.1 Proposal
> 說明：本文件用於收錄正式版發布前之 Gameplay 提案，經確認後再納入正式版本。

---

# 提案一：隱藏互動選項（⑤）

## 說明

1. 遊戲預設互動選項：
   - ①～③：一般劇情選項
   - ④：自由行動
   - ⑤（隱藏）：特殊觸發選項

2. ⑤（隱藏）平時不顯示，僅於符合指定條件時出現。

3. 觸發條件包含但不限於：
   - 墨羽身分
   - 公開身分
   - 年齡
   - 能力
   - 聲望
   - 人物關係
   - 勢力關係
   - 劇情旗標
   - 世界事件
   - 真實歷史事件
   - 歷史戰役
   - 地點
   - 時間線
   - Story Bible 或官方資料庫定義之條件

4. 可觸發：
   - 特殊劇情
   - 隱藏事件
   - 歷史事件參與
   - 戰役參與
   - 特殊人物相遇
   - 特殊勢力互動
   - 特殊任務

5. 必須遵守：
   - Navigation.md
   - Story Bible
   - Official Character Files
   - World Database
   - Official Patch Notes
   - 不得違反官方 Canon。

6. 所有選項格式固定：
   - 每個選項皆必須包含「風險」。
   - 每個選項皆必須包含「備註」。
   - 「風險」與「備註」為固定欄位，不得因一般選項或隱藏選項而改變，也不得省略。

# 提案二：互動選項（①～③）人格／執念提示

## 說明

1. 劇情互動選項 ①～③ 於產生時，必須比對：

   MC-001_墨羽_Official_Character_File_v3.0.1

2. 若該選項符合墨羽 Official Character File 所定義之：
   - 真實人格
   - 執念

   則於「備註」中加入對應提示。

3. 提示內容應直接標示符合的人格或執念，例如：
   - 符合善良
   - 符合責任感
   - 符合初心
   - （後續依 Official Character File 擴充）

4. 風險與備註為固定欄位，不得省略。

5. 標示僅作為角色扮演提示，不代表最佳選項，亦不得影響劇情演算、公平性或玩家自由選擇。

---

## 顯示格式

① 主動扶起受傷的老人。  
風險：低｜備註：符合善良，可能影響人物關係與後續事件。

② 主動接下聚落分配的工作。  
風險：低｜備註：符合責任感，可能影響人物關係與後續事件。

③ 留下照顧孩童，讓其他大人前往警戒。  
風險：低｜備註：符合初心，可能影響人物關係與後續事件。
