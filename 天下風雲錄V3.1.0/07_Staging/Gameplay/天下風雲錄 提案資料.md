# 天下風雲錄 Gameplay 提案資料

> 狀態：Staging / Raw Proposal Data
> 用途：收錄使用者在遊戲中即時提出的原始 Gameplay 提案、測試想法與回饋。
> Canon Status：Non-Canon / Staging Only

---

## 定位

本文件是「原始提案資料庫」。

每當使用者在遊戲中提出新的玩法、互動、系統、選項、顯示格式、規則調整或測試想法時，AI 必須即時將內容追加至本文件。

本文件只負責保存原始資料與第一層整理，不負責最終版本決策。

---

## 寫入規則

```text
1. 新提案一律 Append，不覆蓋舊提案。
2. 每筆提案必須有 Proposal ID。
3. 每筆提案必須保留使用者原意。
4. 若提案尚未完整，仍可先寫入 Draft，後續再補充。
5. 若提案來自遊戲實測，必須記錄當時章節、場景與觸發原因。
6. 本文件內容不得直接成為 Canon。
7. 本文件內容需由 `天下風雲錄 提案彙整.md` 彙整後，才可進入版本評估。
```

---

## Proposal ID 規則

```text
格式：GP-YYYYMMDD-NNN

GP = Gameplay Proposal
YYYYMMDD = 建立日期
NNN = 當日流水號，從 001 開始
```

---

Scene Context: 通用互動選項系統
Scope: Gameplay / Interaction / UI

### 使用者原始提案

建立隱藏互動選項（⑤），平時不顯示，只有在符合特定條件時才出現。

### 初步整理

遊戲預設互動選項：
- ①～③：一般劇情選項
- ④：自由行動
- ⑤（隱藏）：特殊觸發選項

⑤ 可依墨羽身分、公開身分、年齡、能力、聲望、人物關係、勢力關係、劇情旗標、世界事件、真實歷史事件、歷史戰役、地點、時間線或官方資料條件觸發。

### 測試想法

在不同章節場景中測試是否能依條件顯示或隱藏第五選項，並確認不破壞 Canon 與玩家自由選擇。

### 即時測試結果

待後續遊戲流程測試。

### 備註

所有選項仍必須包含「風險」與「備註」。

Scene Context: 通用互動選項系統
Scope: Gameplay / Interaction / Roleplay Hint
Canon Status: Non-Canon / Staging Only

劇情互動選項 ①～③ 產生時，比對墨羽 Official Character File。若選項符合墨羽真實人格或執念，於備註中加入提示。

提示內容可標示為：
- 符合善良
- 符合責任感
- 符合初心

### 測試想法

在每次產生互動選項時，比對 MC-001 墨羽 Official Character File，確認提示是否自然、清楚且不造成選項誘導。

### 即時測試結果

待後續遊戲流程測試。

### 備註

風險與備註為固定欄位，不得省略。

## GP-20260705-003：Proposal Capture Automation

Status: Accepted
Created: 2026-07-05
Updated: 2026-07-05
Source: 使用者指示
Scene Context: Staging Workflow
Scope: Gameplay / Workflow / Staging
Canon Status: Non-Canon / Staging Only

### 使用者原始提案

建立一套自動化流程：使用者每當在遊戲中提出新的 Gameplay 提案、進行測試、修訂提案或確認測試結果時，AI 必須即時寫入提案檔。

### 初步整理

提案資料與提案彙整應拆分：
- 提案資料：保存遊戲中提出的原始提案。
- 提案彙整：整理、分類、追蹤測試狀態與版本採用可能性。

### 測試想法

之後每次使用者於遊戲中提出新提案或測試結果時，AI 依規則生成 Proposal ID、追加提案、更新狀態並回報結果。

### 即時測試結果

已建立拆分規則，待後續使用。

### 備註

此自動化本身屬於 Staging Workflow，不屬於正式 Canon。
