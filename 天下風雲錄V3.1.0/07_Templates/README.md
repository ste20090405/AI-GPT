# 07_Templates

Templates 用於保存《天下風雲錄》所有正式 Markdown 文件模板。

---

## 職責

- 人物檔模板
- 勢力檔模板
- 城鎮檔模板
- 武學檔模板
- 事件檔模板
- Patch Notes 模板
- Game Data 模板
- Changelog 模板

---

## 規則

1. 模板只提供格式，不保存正式設定。
2. 所有正式檔案應盡量依模板建立。
3. 模板修改需同步檢查既有正式檔案是否需要調整。
4. 模板不得與 Story Bible 或 Project Rules 衝突。
