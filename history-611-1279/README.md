# 611–1279 中國中古至宋元之際歷史資料庫

本資料庫整理西元 611 年（隋煬帝大業七年）至西元 1279 年（南宋崖山海戰）之間的歷史事件、制度變遷、戰爭、人物、文化、宗教、科技、外交，以及可考的野史／筆記材料。

> 注意：此專案採「逐批建置」方式。668 年跨度極大，若納入正史、野史、筆記、地方志與專題研究，完整資料量可能達數萬條事件。此版本先建立 Markdown 架構與第一批年度資料。

## 時間範圍

- 起點：西元 611 年，隋煬帝大業七年
- 終點：西元 1279 年，南宋崖山海戰，宋亡

## 收錄原則

每則事件盡量包含：

- 年份與朝代
- 事件類型
- 事件摘要
- 背景
- 影響
- 主要人物
- 可對應史料
- 是否屬正史、野史、筆記或後世傳說

## 主要分期

1. 611–618：隋末動亂與唐朝建立
2. 618–690：唐初政治整合與貞觀、永徽之治
3. 690–705：武周時期
4. 705–755：盛唐政治、邊疆與文化高峰
5. 755–763：安史之亂
6. 763–907：中晚唐、藩鎮、宦官與黃巢之亂
7. 907–960：五代十國
8. 960–1127：北宋、遼、西夏並立
9. 1127–1234：南宋、金、西夏與蒙古興起
10. 1234–1279：蒙古／元與南宋滅亡
