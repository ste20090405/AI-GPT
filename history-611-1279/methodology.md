# 整理方法

## 事件分級

本資料庫將事件暫分為四級：

- A 級：政權更替、重大戰爭、制度改革、外交格局轉折。
- B 級：重要人物任免、軍事行動、災異、經濟政策、宗教與文化事件。
- C 級：地方事件、人物逸聞、筆記材料、民間傳說。
- D 級：待考材料、異說、後世附會或僅見於野史者。

## 史料類型

- 正史：如《隋書》《舊唐書》《新唐書》《舊五代史》《新五代史》《宋史》《遼史》《金史》《元史》等。
- 編年史：以《資治通鑑》為核心。
- 政書與制度史：如《通典》《文獻通考》等。
- 筆記與野史：如《隋唐嘉話》《朝野僉載》《酉陽雜俎》《太平廣記》《北夢瑣言》《夢溪筆談》《夷堅志》等。
- 地方與城市記錄：如《東京夢華錄》《武林舊事》等。

## 標記方式

每一事件盡量標示：

```yaml
年: 611
朝代: 隋
年號: 大業七年
類型: 政治 / 軍事 / 外交 / 制度 / 文化 / 宗教 / 科技 / 災異 / 野史
可信度: A / B / C / D
來源: 《資治通鑑》卷一八一；《隋書》相關本紀或列傳
```

## 野史處理原則

野史、筆記與傳說不與正史混同。若事件僅見於筆記，會標為「筆記／野史」或「待考」。若同一事件在正史與野史中有不同敘述，會分列「正史說」與「異聞」。
