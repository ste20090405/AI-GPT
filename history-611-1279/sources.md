# 主要史料與參考來源

## 正史與編年史

- 《隋書》：隋代本紀、列傳、志，是隋末政治與制度的核心材料。
- 《北史》：保存北朝至隋相關人物與制度材料。
- 《舊唐書》：唐代較早的正史材料。
- 《新唐書》：宋代重修唐史，制度志與列傳材料豐富。
- 《資治通鑑》：編年體核心參考，611–959 年尤其重要。
- 《舊五代史》：五代政權史的重要材料。
- 《新五代史》：歐陽修重修五代史。
- 《宋史》：北宋與南宋制度、人物、列傳與志的主要材料。
- 《遼史》《金史》《元史》：遼、金、元與宋遼金元關係的重要材料。

## 使用注意

1. 正史不等於完全客觀，仍需交叉比對。
2. 野史與筆記可保存社會風俗、怪異傳聞、城市生活與人物逸聞，但可信度需分級。
3. 同一事件若涉及多政權，例如宋、遼、金、西夏、蒙古，應同時參看雙方或多方史料。
4. 年月日若存在換算問題，先保留傳統紀年，再附西元年。
